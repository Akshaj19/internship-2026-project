# Motion Gesture Recognition Dashboard

## Project Overview
This project is a web-based Motion Gesture Recognition Dashboard developed during my internship. The system uses smartphone accelerometer sensor data to detect and classify user gestures in real time.

## Features
- Real-time accelerometer data collection
- Gesture detection using Edge Impulse
- Live prediction dashboard
- Sensor status monitoring
- Real-time X, Y, Z axis visualization
- Analytics and performance tracking
- CSV data export
- Mobile and desktop responsive UI
- Firebase Hosting deployment

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Edge Impulse
- Firebase Hosting
- VS Code

## Gestures Trained
- Clockwise Circle
- Anti-Clockwise Circle
- Wave
- Idle

## What I Learned
- Sensor data acquisition using smartphone accelerometers
- Edge Impulse data collection and model training
- Frontend development with HTML, CSS, and JavaScript
- Responsive UI/UX design
- Firebase Hosting deployment
- Real-time data visualization and gesture classification

## Deployment
The project was deployed using Firebase Hosting and can be accessed through a secure HTTPS connection.
